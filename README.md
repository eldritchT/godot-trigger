# Trigger
A simple implementation of triggers that call a method when entered/exited.

## Usage
1. Install the addon by dragging the `addons` folder into your project and enabling it in the `Project -> Project Settings... -> Plugins`.
2. Go to scene you want to add triggers into, then create a new Trigger2D/Trigger3D node and set it up, there is information in the tooltips.
